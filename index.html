Habilidades Tecnológicas

HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar páginas web. Utiliza etiquetas para definir elementos como texto, imágenes y enlaces, indicando al navegador cómo mostrar el contenido.

CSS

Cascading Style Sheets es un lenguaje de diseño utilizado para definir la presentación y el aspecto visual de documentos HTML.

Java Script

Es un lenguaje de programación que se utiliza principalmente para crear contenido web interactivo y dinámico. También te permite agregar funcionalidades como animaciones, validaciones y DOM (Document Object Model).

Bootstrap

Es un framework front-end de código abierto que facilita el desarrollo visual de los sitios web y aplicaciones web responsivas y móviles.

Python

Es un lenguaje de programación de alto nivel y fácil de aprender, utilizado para desarrollo web, análisis de datos, inteligencia artificial y más.

Java

Es un lenguaje de programación de alto nivel, orientado a objetos, conocido por su portabilidad y uso en aplicaciones web, móviles y empresariales.
